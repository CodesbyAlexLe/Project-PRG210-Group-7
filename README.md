# Grocery Store Management System

The Grocery Store Management System is a console-based application designed to streamline grocery store operations. It provides two separate interfaces:

Manager Interface – for managing departments and products
Customer Interface – for browsing items and making purchases

The system demonstrates key Object-Oriented Programming (OOP) concepts such as inheritance, polymorphism, and dynamic memory management
> ------------------------------------------------------------------------------------------------------------------------- <
# Features:

Manage Interface
- List all departments
- Add new departments
- Add items to departments
- save data to a CSV file

Customer Interface
- Browse Departments
- View items in each department
- Add items to cart
- View cart
- Check out and clear cart
> ------------------------------------------------------------------------------------------------------------------------- <
# Concepts used
- OOP
- Inheritance and Polymorphism
- Dynamic memory Allocation
- File handling
